Weekly Meal Menu
Description

A simple Python console app that suggests a meal based on the selected day of the week and meal type (breakfast, lunch, or dinner).

How It Works

User selects a day (1–7).

User selects a meal (1–3).

The program prints a dish name and cooking instructions.

Runs in a loop for repeated use.

Classes

User_Input – gets and validates user input.

Menu – stores meals and returns the selected recipe.
